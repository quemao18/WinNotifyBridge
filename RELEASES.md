# Releases

This file provides a simple release history for WinNotifyBridge.

## v1.1.0

Feature and installer update release.

### Included

- New keep-awake setting for better away-from-keyboard reliability
- Generic diagnostics monitor with Event Viewer and WPN Database views
- Tray icon improvements
- Local listener path override support for development setups
- MSI installer updated to launch the tray after installation

### Download

Download the `WinNotifyBridge-v1.1.0.msi` asset from the GitHub Releases page.

## v1.0.0

Initial public release.

### Included

- Windows Service
- Tray application
- Notification listener
- Telegram forwarding
- Diagnostics monitor for troubleshooting
- MSI installer for end-user installation
